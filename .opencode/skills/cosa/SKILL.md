## What I do

- Install coreos-assembler (cosa) from source
- Test qcow2 images with `cosa run` or `kola`
- Run integration tests with kola

## Prerequisites

- Fedora/RHEL system with dnf
- Access to /dev/kvm for virtualization
- Sufficient disk space for builds

## Installation

> **Note**: Installation is only required if `cosa` is not already in your PATH.
> Check with: `which cosa || echo "cosa not found"`

### 1. Install dependencies

```bash
sudo dnf install -y git-core make golang qemu-img swtpm qemu-system-x86_64
```

### 2. Clone and build cosa

```bash
git clone https://github.com/coreos/coreos-assembler/ $WORKDIR/coreos-assembler
cd $WORKDIR/coreos-assembler
make
sudo make install
cd -
```

## Usage

### Run a qcow2 image interactively

```bash
cosa run --qemu-image /path/to/image.qcow2
```

## Testing with kola

### Run basic tests on a qcow2 image

```bash
kola run --qemu-image /path/to/image.qcow2
```

### Run specific tests

```bash
kola run --qemu-image /path/to/image.qcow2 basic
```

### List available tests

```bash
kola list
```

## When to use me

Use this skill when:
- You need to test qcow2 images generated by image-builder-cli (ibc)
- You want to run FCOS/bootc images in a VM for testing
- You need to run kola integration tests
